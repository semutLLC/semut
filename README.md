# Semut

Semut is a framework and collection of projects exploring how coordination emerges from recorded actions.

At its core is the observation that large-scale systems—communities, organizations, markets, and civilizations—are built upon accumulated traces of activity. These traces form memories, memories enable knowledge, knowledge shapes norms, norms are preserved by institutions, and institutions support ideologies.

The Semut Layer Model describes this progression:

```text
Trace → Memory → Knowledge → Norm → Institution → Ideology
```

Semut aims to make these layers visible, preserve their evolution over time, and explore practical systems built upon them.

---

## Vision

Most modern systems record traces:

* Transactions
* Messages
* Tasks
* Code commits
* Reviews
* Conversations

However, these traces are often fragmented across many tools and organizations.

Semut explores how these traces can be connected into a larger understanding of coordination and collective behavior.

The goal is not centralized control.

The goal is visibility, memory, and informed coordination.

---

## Core Concepts

### Trace

Observable actions and events.

Examples:

* Completed tasks
* Purchases
* Messages
* Code commits
* Sensor readings

### Memory

Persistent records of traces.

Examples:

* Histories
* Logs
* Repositories
* Reputation systems

### Knowledge

Patterns extracted from memory.

Examples:

* Best practices
* Procedures
* Documentation
* Models

### Norm

Repeated expectations and behaviors.

Examples:

* Team habits
* Community customs
* Professional standards

### Institution

Structures that preserve and reproduce norms.

Examples:

* Businesses
* Governments
* Schools
* Open-source communities

### Ideology

Shared beliefs and narratives that guide institutions.

Examples:

* Missions
* Philosophies
* Cultural values

## Long-Term Goal

Semut seeks to become a practical framework for understanding and improving coordination systems.

Potential applications include:

* Local business networks
* Reputation systems
* Knowledge management
* Community coordination
* Human-AI collaboration
* Open-source ecosystems

By connecting traces to higher-order structures, Semut aims to provide tools for observing how complex systems emerge, adapt, and sustain themselves through time.
